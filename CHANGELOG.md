# Changelog - Mise à Jour Routine Pharma Expert J90

## Version 2.1.0 - Routine Structurée Complète

### 📋 Résumé des Modifications

Mise à jour complète de l'application avec une nouvelle routine journalière et hebdomadaire structurée, adaptée à la situation réelle d'un étudiant en master pharmaceutique avec cours de 8h à 13h.

### ✅ Modifications Apportées

#### 1. **Routine Quotidienne Détaillée** (`js/script.js`)

La routine journalière a été réorganisée en **10 blocs horaires** pour maximiser la compréhension et la progression académique :

- **06:00 - 06:15** : Réveil & Hydratation
  - Hydratation, préparation, éviter le téléphone
  
- **06:15 - 06:45** : Prélecture Scientifique (30 min)
  - Titres, concepts, mots-clés du cours du jour
  - Objectif : préparer le cerveau avant le cours
  
- **06:45 - 07:15** : Révision J-1 (30 min)
  - Mini fiche synthèse, correction des concepts
  - Méthode : écrire une fiche avec concepts clés, mécanismes, paramètres
  
- **07:15 - 07:40** : Préparation Université (25 min)
  - Transport, matériel, concentration
  
- **08:00 - 13:00** : Cours Master (5h)
  - Prise de notes scientifique active
  - Noter : concepts importants, mécanismes, équations, exemples
  - Toujours écrire : Pourquoi ce phénomène se produit ?
  
- **14:30 - 16:00** : Bloc Principal : Apprentissage (1h30)
  - **Le bloc le plus important de la journée**
  - Étapes :
    1. Relire les notes (15 min) - identifier parties floues
    2. Compréhension scientifique (45 min) - polycopié, ouvrage, schémas
    3. Reconstruction active (30 min) - écrire de mémoire concepts et mécanismes
  
- **16:00 - 16:30** : Pause Repos (30 min)
  - Repos complet et recharge
  
- **16:30 - 17:30** : Approfondissement Scientifique (1h)
  - Lecture d'articles scientifiques
  - Lecture d'ouvrages
  - Compréhension de modèles biopharmaceutiques
  - Objectif : développer une culture scientifique au-delà du cours
  
- **20:00 - 21:00** : Analyse de Données : R (1h)
  - Manipulation de données
  - Visualisation
  - Régression
  - Analyse de variance (ANOVA)
  - Important : relier l'analyse aux problématiques pharmaceutiques
  
- **21:00 - 21:20** : Organisation & Planification (20 min)
  - Classement des notes
  - Organisation des fichiers
  - Planification du lendemain

#### 2. **Routine Hebdomadaire Structurée** (`index.html`)

Chaque jour a un focus spécifique pour optimiser l'apprentissage :

- **Lundi** : Compréhension des Cours
  - Révision approfondie des cours
  
- **Mardi** : Mécanismes Scientifiques
  - Schémas conceptuels
  
- **Mercredi** : Lecture Scientifique
  - Objectif : lire 1-2 articles scientifiques
  
- **Jeudi** : Devoirs & TD
  - Exercices
  - Travaux dirigés
  - Préparation des évaluations
  
- **Vendredi** : Analyse de Données : R
  - Pratique statistique
  
- **Samedi (2-3h)** : Bloc Stratégique
  - Synthèse des cours de la semaine
  - Création de fiches scientifiques
  - Révision des mécanismes
  
- **Dimanche (1h)** : Planification
  - Identifier les points difficiles
  - Préparer la semaine

#### 3. **Indicateurs de Progression Hebdomadaire** (`index.html`)

Nouveaux indicateurs pour suivre les objectifs hebdomadaires :

- **Cours Révisés** : 100% (Objectif : tous les cours)
- **Articles Lus** : 2-3 (Objectif : Mercredi)
- **Heures R & Stats** : 4-5h (Objectif : Vendredi)
- **Pages de Notes** : 10-15 (Objectif : Notes scientifiques)

#### 4. **Système de Fiches Recommandé**

Chaque cours doit produire une fiche contenant :
- Définition du phénomène
- Mécanisme scientifique
- Paramètres influents
- Modèle mathématique éventuel
- Application pharmaceutique

Ce type de fiche correspond à la pensée d'un chercheur académique.

### 🎯 Objectifs Principaux
