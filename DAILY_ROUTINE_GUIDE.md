# Guide Complet : Routine Journalière Interactive

## 📋 Vue d'ensemble

La **Routine Quotidienne Structurée** est le cœur de votre application Pharma Expert J90. Elle vous permet de suivre précisément chaque bloc de votre journée d'étudiant en master pharmaceutique, de valider vos tâches complétées et de chronométrer vos sessions.

## 🚀 Comment Accéder à la Routine Journalière

### Via le Menu de Navigation

1. **Sur Desktop** : Cliquez sur le bouton **"Planning"** dans la barre de navigation gauche
2. **Sur Mobile** : Appuyez sur l'icône **"Planning"** dans la barre de navigation inférieure

La section Planning s'affichera avec :
- **Titre** : "Routine Quotidienne Structurée"
- **Sous-titre** : Informations sur vos cours (8h-13h) et votre spécialité
- **Badge** : "Objectif : Chercheur Académique"

## 📅 Structure de la Routine Journalière

Votre journée est divisée en **10 blocs horaires** optimisés pour maximiser votre apprentissage :

### 1️⃣ Réveil & Hydratation
- **Horaire** : 06:00 - 06:15 (15 min)
- **Objectif** : Préparation mentale et physique
- **Actions** : Hydratation, éviter le téléphone, se préparer

### 2️⃣ Prélecture Scientifique
- **Horaire** : 06:15 - 06:45 (30 min)
- **Objectif** : Préparer votre esprit aux cours
- **Actions** : Lire les titres, concepts et mots-clés du cours

### 3️⃣ Révision J-1
- **Horaire** : 06:45 - 07:15 (30 min)
- **Objectif** : Renforcer la mémoire à long terme
- **Actions** : Mini fiche synthèse, correction des concepts mal compris

### 4️⃣ Préparation Université
- **Horaire** : 07:15 - 07:40 (25 min)
- **Objectif** : Arriver à l'université préparé
- **Actions** : Transport, vérifier le matériel, se concentrer

### 5️⃣ Cours Master
- **Horaire** : 08:00 - 13:00 (300 min / 5h)
- **Objectif** : Apprentissage actif en classe
- **Actions** : Prise de notes scientifique active, participation

### 6️⃣ Bloc Principal : Apprentissage
- **Horaire** : 14:30 - 16:00 (90 min)
- **Objectif** : Consolidation des apprentissages
- **Actions** : Révision → Compréhension → Reconstruction

### 7️⃣ Pause Repos
- **Horaire** : 16:00 - 16:30 (30 min)
- **Objectif** : Recharge mentale
- **Actions** : Repos complet, détente, hydratation

### 8️⃣ Approfondissement Scientifique
- **Horaire** : 16:30 - 17:30 (60 min)
- **Objectif** : Aller au-delà du cours
- **Actions** : Lire articles scientifiques, ouvrages, modèles biopharmaceutiques

### 9️⃣ Analyse de Données : R
- **Horaire** : 20:00 - 21:00 (60 min)
- **Objectif** : Maîtriser l'analyse statistique
- **Actions** : Manipulation de données, visualisation, régression, ANOVA

### 🔟 Organisation & Planification
- **Horaire** : 21:00 - 21:20 (20 min)
- **Objectif** : Préparer le lendemain
- **Actions** : Classer les notes, organiser les fichiers, planifier J+1

## ✅ Comment Valider une Tâche

### Méthode 1 : Clic sur la Carte Entière
1. Cliquez **n'importe où** sur la carte de la tâche
2. Le cercle blanc se transforme en **cercle vert avec checkmark**
3. La tâche apparaît **légèrement transparente** (grayscale)
4. Vous gagnez **+10 XP**

### Méthode 2 : Clic sur le Cercle
1. Cliquez spécifiquement sur le **cercle blanc** à gauche
2. Même effet que la méthode 1
3. Vous pouvez annuler en cliquant à nouveau

### Indicateurs Visuels de Validation
- ✅ **Cercle vert** : Tâche complétée
- ⭕ **Cercle blanc** : Tâche non commencée
- 🎨 **Grayscale** : Tâche complétée (désaturée)
- 📊 **Barre de progression** : Affichée en haut du Planning

## ⏱️ Utiliser le Minuteur pour Chaque Session

### Démarrer un Minuteur

1. Repérez le bouton **"Minuteur"** (avec icône play) à droite de la tâche
2. Cliquez sur le bouton
3. Le minuteur démarre automatiquement avec le compte à rebours

### Affichage du Minuteur Actif

Quand un minuteur est en cours :
- La carte de la tâche est **mise en évidence avec un anneau vert**
- Le temps restant s'affiche en **MM:SS** (ex: 30:45)
- La durée totale est affichée en dessous (ex: 90 min)
- Deux boutons remplacent le bouton "Minuteur" :
  - ⏸️ **Pause** : Met le minuteur en pause
  - ⏹️ **Stop** : Arrête complètement le minuteur

### Mettre en Pause

1. Cliquez sur le bouton **⏸️ Pause**
2. Le minuteur s'arrête mais conserve le temps restant
3. Vous pouvez reprendre plus tard

### Reprendre une Session en Pause

1. Cliquez à nouveau sur le bouton (qui affiche maintenant ▶️ Reprendre)
2. Le minuteur continue à partir du temps restant

### Arrêter une Session

1. Cliquez sur le bouton **⏹️ Stop**
2. Le minuteur s'arrête et réinitialise
3. Vous pouvez relancer une nouvelle session

### Récompenses pour Session Complétée

À la fin d'une session :
- 🔔 **Notification système** (si autorisée)
- 🔊 **Son de notification** (bip)
- ✨ **Message dans l'app** : "Session terminée : [Nom] ! +15 XP"
- 📈 **+15 XP** automatiquement accordés
- 🎮 **Interface mise à jour** avec votre nouveau niveau/XP

## 🎯 Conseils pour Optimiser Votre Routine

### ✨ Meilleures Pratiques

1. **Respectez les horaires** : Les blocs sont optimisés pour votre apprentissage
2. **Utilisez le minuteur** : Cela vous aide à rester concentré
3. **Validez au fur et à mesure** : Ne pas attendre la fin de la journée
4. **Prenez la pause repos** : Elle est essentielle pour la performance
5. **Organisez chaque soir** : Le bloc final prépare votre succès de demain

### 📱 Utilisation sur Mobile

- Les boutons s'adaptent à la taille de l'écran
- Le texte "Minuteur" se cache sur petit écran (seule l'icône play reste)
- Vous pouvez scroller horizontalement pour voir tous les contrôles

### 🌙 Gestion de l'Écran

Pendant une session chronométrée :
- L'écran **ne se mettra pas en veille** (Wake Lock API)
- Vous pouvez utiliser votre appareil sans interruption
- L'écran se déverrouille automatiquement à la fin de la session

## 🐛 Dépannage

### Le minuteur ne démarre pas
- Vérifiez que JavaScript est activé dans votre navigateur
- Rechargez la page et réessayez
- Vérifiez la console (F12) pour les erreurs

### Les tâches ne se sauvegardent pas
- Vérifiez que le stockage local (localStorage) est activé
- Assurez-vous que vous n'êtes pas en mode navigation privée
- Videz le cache et rechargez

### Les notifications ne s'affichent pas
- Allez dans les paramètres de votre navigateur
- Autorisez les notifications pour ce site
- Redémarrez votre navigateur

### Le minuteur continue après fermeture
- C'est normal ! Le minuteur utilise le temps système
- Si vous fermez l'app, le minuteur continue en arrière-plan
- Rouvrez l'app pour voir le temps restant
